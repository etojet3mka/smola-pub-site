# СМОЛА.паб — сайт для курсового проекта

Статический сайт для курсовой работы по предмету «Основы web-разработки и web-дизайна».
Тема теоретической части: «Кросс-браузерность и кросс-платформенность: почему это важно для бренда».

## Структура

- `index.html` — главная страница
- `menu.html` — меню
- `events.html` — афиша
- `about.html` — описание бренда
- `booking.html` — форма бронирования
- `css/style.css` — стили
- `js/app.js` — мобильное меню, фильтр меню, валидация формы, демо-аналитика
- `img/` — SVG-изображения
- `docs/test-checklist.md` — чек-лист тестирования

## Как открыть локально

1. Распакуйте архив.
2. Откройте файл `index.html` в браузере.
3. Для разработки лучше открыть папку в VS Code и запустить Live Server.

## Как опубликовать на Netlify Drop

1. Перейдите на `app.netlify.com/drop`.
2. Перетащите всю папку `smola_pub_site` в окно загрузки.
3. Получите ссылку на сайт.
4. Проверьте главную страницу, меню, форму и мобильную версию.

## Что уже реализовано

- Адаптивная вёрстка.
- 5 HTML-страниц.
- Рабочее мобильное меню.
- Форма бронирования с проверкой полей.
- Фильтр по меню.
- Демо-аналитика через `console.log`.
- Семантические HTML5-теги.
- Относительные пути к файлам.
- SVG-изображения без внешних зависимостей.


## Шапка сайта

Шапка сделана светлой, с центральным логотипом. Пункты меню слева и справа выровнены симметрично. При прокрутке вниз шапка скрывается, при прокрутке вверх снова появляется.


## Экран 18+

Экран подтверждения возраста появляется один раз при заходе на сайт в рамках текущей вкладки. После подтверждения пользователь может переходить по страницам без повторного окна. После закрытия вкладки проверка появится снова. Кнопка «Мне нет 18» блокирует доступ к основному контенту.
